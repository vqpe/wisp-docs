# Server Eggs

## What is a Server Egg?
Server Eggs are a preconfigured template that sets up the environment (like Node.js or Python) and startup commands.

Wispbyte supports the following server eggs:

### Free
Python
NodeJS
Bun (NodeJS)
Java
C#
Rust
Lua (Luvit)
MongoDB
MariaDB
Redis

### Paid
Python
NodeJS
More per request
